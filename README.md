# Tarun Prakash - Personal Website

My personal website built with Jekyll, based on the [AcademicPages](https://github.com/academicpages/academicpages.github.io) template.

**Live site:** [tarunprakash2468.github.io](https://tarunprakash2468.github.io)

## Running Locally

1. Install dependencies:
   ```bash
   bundle install
   ```

2. Start the local server:
   ```bash
   bundle exec jekyll serve -l -H localhost
   ```

3. Visit `localhost:4000`

## Using Docker

```bash
docker compose up
```

Then visit `localhost:4000`.
